CoolBox
=======

Jupyter notebook based genomic data visulization toolkit.
---------------------------------------------------------

![](demo/imgs/Cover.png)

Use [pyGenomeTracks](https://github.com/deeptools/pyGenomeTracks) as plot system.

* Multi-omics data visulization
* GUI widget
* User-friendly API
* Web Access
* Compatible with Jupyter ecosystem
